## Installation

1. Es necesario tener docker y docker compose instalado en la maquina.
2. Abrir la consola en la carpeta donde se encuentra el proyecto, en este caso, la carpeta **tenpo**
3. Correr el siguiente comando
```sh
docker compose up
```
